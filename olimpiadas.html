Olimpiadas O jogo

Isso se consiste em um jogo sozinho que foi criado por mim, a bases de textos, para destacar a vida do lutador ohmid ahmadisafa, tendo uma historia curta, mas bem trabalhada, o quadro foi feito com base e inspiração na luta de omid para conseguir ser o lutador que ele é hoje, sendo o jogo uma homangem a ele

Linguagems utilizadas: CSS e Java script
